Main additions

Content from newer versions:

  • Grass Path
  • Colored glass and glass panes
  • Colored beds
  • Spruce and birch doors, fences, and trapdoors
  • Missing items added to the Creative inventory
  • Flat world type
  • Sprinting!

New features:

  • Reworked texture loading system: instead of using terrain-atlas.tga, textures are loaded similarly to Java Edition, directly from folders
  • World editing: ability to rename worlds and change game modes
  • Server editing: ability to modify server information without recreating the server
  • Dye coloring system for wood, planks, wooden stairs, slabs, and fences
  • Dye coloring system for bricks, brick stairs, and brick slabs
  • Dye coloring system for sign text
  • Ability to change chat message colors and backgrounds

...and much more.
