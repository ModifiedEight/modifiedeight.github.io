Releases


1.4.0 June 23, 2026

* The client has been split into two versions: Classic and New Additions. The Classic version includes new features, while New Additions includes both the new features and the new blocks and items.
* Added the ability to use the new world generation when creating a world.
* Added auto-jump.
* Added an FOV toggle.
* Added a Discord integration toggle in the settings.
* Fixed Marketplace-related issues.
* Added the camera item to the creative inventory.

1.3.0 June 16, 2026

* The client now has 2 modes: Modded and New Additions. The Modded mode adds new functionality, while New Additions adds new blocks and items. If you choose the Modded mode, you will have access to all new features except those related to new blocks and items.
* Added an option to make the old world type infinite in the world settings.
* Added an option to change the player's position coordinates in the world settings.
* Added an FPS counter.
* Added a Debug Screen.
* Added the ability to enable and disable fog.
* The render distance can now be increased even further.
* Added a switch between the old texture loading system (using the terrain-atlas) and the new one.
* Added a Marketplace: integrated downloading of maps and textures from https://mcpe-source.github.io/, as well as a map and texture manager. In the world manager, you can import worlds from archives, and in the texture manager, you can save and install textures from archives.
* Added Discord Rich Presence integration.
* Added the iron door to the creative inventory.
* Updated the colored bricks texture (thanks to mryoshi).
* Other changes

1.1.0 June 9, 2026

* Added a world time editor to the World Settings screen, along with the ability to freeze time.
* Added an option to enable or disable sprinting.
* Added a person view toggle button next to the chat button.
* Added ore ingots to the creative inventory.
* Potentially fixed a crash on Windows that occurred when clicking the Create World, Edit World, or Server buttons.
* Added support for rapid character input and deletion when holding down keys, instead of requiring repeated key presses.
* Fixed an issue where players could not enter a doorway if a path block was placed in front of it.
* Fixed an issue where flight and damage immunity remained active after switching to Survival mode.
* Fixed the armor slot textures in the inventory.
* Fixed an issue that prevented players from sleeping in beds.
* Fixed an issue where players spawn high above the ground when creating a Flat world.
* Improved the layout of elements on the world creation screen for mobile devices.
* Other changes and bug fixes.
