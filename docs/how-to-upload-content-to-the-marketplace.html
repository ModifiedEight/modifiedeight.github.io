How to upload content to the marketplace

ModifiedEight uses the MCPE-Source platform to obtain maps and textures. This platform allows users to upload various content for Minecraft PE, using their GitHub account for authentication.
At the moment, MCPE-Source only has a mobile application available: https://github.com/MCPE-Source/MCPE-Source-Android
